# 🚀 **Sevancio — Cara Run di Local PC**

## 📋 Prerequisites

Sebelum mulai, pastikan PC lo punya:

| Tools | Minimal Versi | Cek |
|-------|---------------|-----|
| **Node.js** | 20+ | `node -v` |
| **npm** | 9+ | `npm -v` |
| **Hermes Agent** | 0.6+ | `hermes --version` |
| **Git** | 2.x | `git --version` |

---

## ⚙️ Step 1 — Clone & Install

```bash
# Clone repo
git clone https://github.com/sevandri/sevancio_agentic.git
cd sevancio_agentic

# Install dependencies
npm install
```

> ⏳ Ini bakal download: React, Electron, MediaPipe WASM, ONNX Runtime, fonts, dll. Kira-kira 1-2 menit tergantung koneksi.

---

## 🔐 Step 2 — Setup Gemini API Key

Buat Gemini API key gratis di **[Google AI Studio](https://aistudio.google.com/apikey)** (Google kasih free tier).

Terus copy file env:

```bash
cp .env.example .env
```

Edit `.env` dan isi:

```bash
GEMINI_API_KEY=AIzaSy...         # <— paste key lo di sini
SEVANCIO_USER_NAME=Adri           # nama panggilan lo
```

---

## 🔗 Step 3 — Enable Hermes API

Sevancio butuh Hermes Agent untuk kerja berat. Pastikan Hermes udah terinstall:

```bash
# Cek Hermes
hermes --version

# Enable API server + set API key
echo 'API_SERVER_ENABLED=true' >> ~/.hermes/.env
echo 'API_SERVER_KEY=sevancio-local-dev' >> ~/.hermes/.env

# Restart gateway
hermes gateway restart

# Test koneksi
curl http://127.0.0.1:8642/health
```

> Kalau return `{"status":"ok"}` atau JSON apapun → Hermes siap.

---

## 🎬 Step 4 — Jalankan Sevancio

### Mode Development (hot-reload)

```bash
npm run dev
```

Ini bakal:
1. Start Vite dev server di `http://127.0.0.1:5173`
2. Launch Electron window
3. **Onboarding wizard** muncul otomatis — isi Gemini key & Hermes config
4. Tekan **W** untuk wake, mulai ngobrol!

### Mode Production

```bash
npm start
```

### Package untuk distribusi

```bash
npm run package:mac    # macOS .app (unsigned)
npm run dist:win       # Windows installer
```

## 🐛 Troubleshooting

### "Electron bridge unavailable"
Jalanin pake `npm run dev`, bukan buka index.html langsung di browser.

### "Hermes not reachable"
```bash
# Cek Hermes gateway
hermes gateway status
# Pastikan API_SERVER_ENABLED=true di ~/.hermes/.env
```

### "No audio / mic not working"
- Permission mic → cek system settings
- WebRTC echo cancellation → pake headphones kalo gema

### Wake word "Hey Sevancio" gak berfungsi
Defaultnya **opt-in**. Enable di Settings → centang "Wake Word". Butuh mic permission.

## 💡 Tips

1. **Demo mode** — Set `SEVANCIO_LOAD_TEST_DATA=true` di `.env` buat explore UI tanpa perlu API key (tekan D buat load data, G buat simulasi handoff)
2. **Ganti voice** — `GEMINI_LIVE_VOICE=Zephyr` bisa diganti: Puck, Charon, Kore, Fenrir, Aoede, dll.
3. **Wake word threshold** — Kalo sering false wake, naikin threshold di `useWakeDetector.ts` line 18 (`DEFAULT_THRESHOLD = 0.15` jadi 0.2)
